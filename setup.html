üöÄ Quick Start

1
Open Terminal/Command Prompt
Navigate to the PostiveVocab directory where you saved these files.
2
Start the Server (Choose One)

Option A: Using Node.js (Recommended)

Option B: Using PHP

Option C: No Server (Fallback)

Just open gallery.html directly in your browser!

3
View the Gallery
